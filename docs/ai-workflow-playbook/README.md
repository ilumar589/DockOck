# AI Workflow Playbook Package

This folder contains the self-contained Itineris workflow package: the main playbook, the presentation views, the supporting diagrams, and the screenshots referenced by the document.

## Start Here

- `executive-summary.html`: fastest presentation-first overview for client or leadership discussions
- `workflow-map.html`: interactive explanation of the end-to-end workflow, including validation and test generation
- `agent-routing.html`: interactive view of ownership, specialist routing, and repair vs maintenance decisions
- `delivery-loop.html`: interactive view of how approved artifacts move through OpenSpec, OpenCode, review, and repair
- `index.md`: full written playbook with rationale, screenshots, runbooks, and detailed workflow sections

## Asset Files

- `ai-workflow-overview.svg`: top-level workflow diagram
- `itineris-validation-loop.svg`: validation and test feedback loop diagram
- `ai-agent-routing.svg`: routing diagram for slice ownership and specialist expansion
- `ai-delivery-loop.svg`: OpenSpec-to-OpenCode delivery loop diagram
- `playbook-*.png`: screenshots embedded in the written playbook

## Notes

- The folder is designed to be self-contained. Relative links between the Markdown file, HTML pages, SVGs, and screenshots should continue to work if this folder is moved as a unit.
- The interactive HTML pages pull the Montserrat font and the Itineris logo from public URLs.
